# Dette er brukerstøtte delen av eksamen.

viktigst av alt må brukeren samtykke for informasjonen som blir lagret, men i tillegg til dette må bedriften må huske og bare samle inn data som er nødvendig, dette kan inkludere navn epost, adresse. Passord til brukeren må krypteres og aldri være i klartekst, og brukeren må ha rett til å slette konto/informasjon. 
Data skal overhode ikke bli delt uten brukerens samtykke, og bedriften må si hvorfor de trenger data.

# veiledning
Steg 1, gå til nettstedet ved å skrive inn domenen
steg 2, klikk på "registrer deg", mi øverst høyre gjørne
steg 3, fyll in Navn, Passord, E-post adresse
steg 4, klikk "Oprett konto på bunnen"
